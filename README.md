# ☘️ St. Patrick's Day Trivia Quiz

A fun, single-page Irish trivia app built for work parties and St. Patrick's Day celebrations.

## Features

- **20 questions** covering Irish history, folklore, geography, culture, food & drink, and language
- **Timed challenge** — the clock starts when you do
- **Escalating time penalties** — each wrong answer adds an increasing time penalty (5s × wrong answer count) to your final time
- **Randomized question order** every game
- **Animated UI** with floating shamrocks, smooth transitions, and festive green & gold styling
- **Results screen** with score, completion time, penalty breakdown, and a personalized verdict

## How to Play

1. Click **Start Trivia**
2. Answer all 20 questions as fast as you can
3. Choose carefully — wrong answers add time penalties!
4. See your final score and total time at the end

## Running Locally

No build step required. Just open `src/index.html` in any browser.

## Tech Stack

- Plain HTML, CSS, and vanilla JavaScript — no dependencies
- Google Fonts (Playfair Display, Lora)
